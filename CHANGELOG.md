## 🎯 使用说明

### 访问流程

1. 打开网站首页
2. 看到密码输入界面
3. 输入密码: `1123`
4. 点击"解锁回忆"或按回车
5. 进入主页，享受美丽的特效

### 修改密码

编辑 `assets/js/main.js` 文件，找到这一行：

```javascript
const correctPassword = '1123';
```

将 `'1123'` 改为你想要的密码即可。

### 调整特效数量

在 `assets/js/main.js` 中修改：

```javascript
// 星星数量
const starCount = window.innerWidth < 768 ? 30 : 60;

// 爱心数量
const heartCount = window.innerWidth < 768 ? 10 : 20;
```

### 关闭某个特效

在 `assets/js/main.js` 的 `DOMContentLoaded` 中注释掉对应行：

```javascript
// 关闭星光特效
// createStars();

// 关闭爱心飘落
// createFallingHearts();
```
