# 音频文件说明

## 需要的音频文件

请将以下音频文件放置在此目录下：

### 1. 背景音乐
- **文件名**: `background.mp3`
- **用途**: 网站背景音乐，会在所有页面持续播放
- **建议**: 选择一首温柔浪漫的纯音乐或情歌
- **时长**: 建议 3-5 分钟以上（会循环播放）

### 2. 录音文件（用于音频页面）
放置在 `assets/audio/` 目录下：

- `1022.m4a` - 10月22日的录音
- `1023.m4a` - 10月23日的录音
- `1024.m4a` - 10月24日的录音
- `1025.m4a` - 10月25日的录音
- `1026.m4a` - 10月26日的录音
- `1027.m4a` - 10月27日的录音
- `1103.m4a` - 11月03日的录音
- `1110.m4a` - 11月10日的录音
- `1111.m4a` - 11月11日的录音

## 临时解决方案

如果暂时没有音频文件，可以：

1. **下载免费背景音乐**：
   - 网站：YouTube Audio Library, Pixabay Music
   - 搜索关键词：romantic, love, piano, gentle

2. **使用在线音频转换**：
   - 如果你有其他格式的音频，可以使用在线工具转换为 mp3/m4a

3. **测试用占位符**：
   - 可以暂时使用任何音频文件重命名为 `background.mp3` 进行测试

## 注意事项

- 确保音频文件大小适中（背景音乐建议 < 10MB）
- 使用 mp3 格式以获得最佳浏览器兼容性
- m4a 格式也支持，但 mp3 兼容性更好
